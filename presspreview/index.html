Judgment 1.0 press preview

As we approach Judgment's full version launch date - May 3rd, 2018 - there is now a press preview version available through steam. This version has all the content of the full version, though we may still do some tweaking and fixing some small bugs.

To play the 1.0 press preview, please follow these instructions:

  1. Right click the game in your Steam library, and select "Properties"
  2. Select the "Betas" tab
  3. Enter the preview access password you received and click the "Check Code" button. You should receive a message that says you now have access to the "press" beta branch.
  4. In the drop down, select the "press - Full Version Press Preview" option.
  5. Click the "Close" button.
  6. Install the game, or wait for Steam to update the installation if it was already installed
After following these instructions, launch the game from Steam, and when the game starts you should see a message saying this is the 1.0 preview version. If instead you get a message that says this is an Early Access version, please follow the instructions again, or Contact Us for assistance.
